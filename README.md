RFM segmentation and churn prediction (Random Forest) to identify high-value and at-risk customers, conducted sentiment analysis on product data using VADER to extract perception insights, and generated LLM-generated strategy recommendations for tailored retention actions per segment.

## Overview
This project aims to provide a holistic approach to understanding customer behavior and predicting future actions, such as churn. It integrates various data science techniques, including exploratory data analysis, data cleaning, RFM (Recency, Frequency, Monetary) segmentation, sentiment analysis, and machine learning for churn prediction. The ultimate goal is to equip businesses with actionable insights and LLM-powered recommendations to enhance customer retention and engagement.

## Features
- **Comprehensive EDA**: In-depth analysis of data distributions, correlations, and potential anomalies.
- **Robust Data Cleaning**: Handling of duplicates, missing values, and outliers to ensure data quality.
- **RFM Segmentation**: Categorization of customers into distinct segments based on their purchasing behavior.
- **Sentiment Analysis**: Evaluation of customer sentiment from textual data (e.g., product descriptions).
- **Churn Prediction**: A machine learning model (Random Forest Classifier) to identify customers at risk of churning.
- **LLM-powered Recommendations**: Integration with a Large Language Model to generate tailored business strategies for different customer segments.

## RFM Segmentation
RFM segmentation is a marketing technique used to quantitatively rank and group customers based on their purchasing habits. The project calculates Recency (days since last purchase), Frequency (total number of purchases), and Monetary (total spend) values for each customer. These are then converted into R, F, and M scores (1-4), which sum up to a total RFM_Score. This score helps identify high-value customers (e.g., Champions) and those at risk.

## Sentiment Analysis
Sentiment analysis is performed on the 'Description' column of the dataset using VADER (Valence Aware Dictionary and sEntiment Reasoner). VADER is a lexicon and rule-based sentiment analysis tool that is specifically attuned to sentiments expressed in social media. It provides a compound score, normalized between -1 (most negative) and +1 (most positive), which is then categorized into 'Positive', 'Negative', or 'Neutral' sentiment.

## Churn Prediction
The churn prediction model uses the RFM metrics as features to predict whether a customer is likely to churn. A 'Churn' label is synthetically created based on Recency (customers inactive for more than 90 days are marked as churned). A Random Forest Classifier is trained on this data, and its performance is evaluated using a classification report, highlighting the model's ability to identify churners accurately. Feature importances help understand which RFM aspects are most indicative of churn.
